Install Snipe-IT Windows 2008 R2 With IIS

Prereq (Install all of these prior to continuing)

  • IIS
  • IIS URL Rewrite
  • PHP 5.4
  • MariaDB 10.0.14 for Windows 64-bit | 32-bit
  • PHP Manager for IIS (makes managing PHP on IIS much easier)
  • [Composer install with the shell menus](https://getcomposer.org/Composer-Setup.exe )
  • [Notepad++](http://www.notepad-plus-plus.org/download/v6.6.6.html ) for editing files

Setting Up an IIS Website

  • Download Snipe-IT
  • Extract to C:\inetpub\wwwroot\snipe-it (folder name can be changed but we will reference it as is shown here)
  • Run IIS Manager
  • Right Click Sites and Add Website
Site name: Snipe IT
Physical path: C:\inetpub\wwwroot\snipe-it\public
Binding
Type: http
IP address: All Unassigned or a specific IP if you have one you will be using
Port: 80 or any you wish to use
Host name: assets.portal.local (this can be changed to suit your needs)
  • Click OK Your site will now appear in the list
  • Double click on your site

  • Double Click URL Rewrite

    1. In the action pane click Import Rules...
    2. Click the ... button
    3. Go to C:\inetpub\wwwroot\snipe-it\public
    4. Select .htaccess file
    5. Click Open then Import

    6. In the action pane click Apply

    7. Type php artisan app:install and follow the prompts
    8. Add permissions for the IIS user for the storage folder:
    9. Goto C:\inetpub\wwwroot\snipe-it\app
    10. Right Click storage -> Properties
    11. Goto Security Tab -> Edit
    12. Click Add and change location to local machine
    13. Type IUSR in object name box
    14. Click OK
    15. Give IUSR full control
    16. Click OK twice
    17. Add permissions for the IIS user for the uploads folder:
    18. Goto C:\inetpub\wwwroot\snipe-it\app
    19. Right Click uploads -> Properties
    20. Goto Security Tab -> Edit
    21. Click Add and change location to local machine
    22. Type IUSR in object name box
    23. Click OK
    24. Give IUSR full control
    25. Click OK twice
