Snipe-IT Documentation

Requirements

  • PHP 5.4 or later
  • MCrypt PHP Extension
  • Fileinfo Extension
  • php-pdo
  • php-mysql
  • php-mbstring
  • MySQL or MariaDB

And one of the following image libraries (for QR codes, asset model image and user avatar image uploads).

  • GD Library (>=2.0) … or …
  • Imagick PHP extension (>=6.3.8)
